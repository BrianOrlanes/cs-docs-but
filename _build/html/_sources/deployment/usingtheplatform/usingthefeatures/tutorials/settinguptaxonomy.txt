=======================
Set Up a Taxonomy
=======================
Taxonomy is a product classification used to organize the content.



How-to: Set Up a Taxonomy
==================================
To do this, follow these steps:

|

* Log in and click on the **Administration** icon.

.. image:: /feature/_images/mdmimages/pxp_settinguptaxonomy_clickadministration.PNG
|
|

* Click on the **Data Model** tab.

.. image:: /feature/_images/mdmimages/pxp_settinguptaxonomy_clickdatamodel.PNG
|
|

* Click on **Taxonomies**.

.. image:: /feature/_images/mdmimages/pxp_settinguptaxonomy_clicktaxonomies.PNG
|
|

* Then, click on **Master Taxonomies**.

.. image:: /feature/_images/mdmimages/pxp_settinguptaxonomy_clickmastertaxonomy.PNG
|
|

* Click on the '+' icon to expand the taxonomy, vertically or horizontally.

.. image:: /feature/_images/mdmimages/pxp_settinguptaxonomy_useplus.PNG
|
|

* Taxonomy window appears. Enter details for the new taxonomy and click on the **Create** button.

.. image:: /feature/_images/mdmimages/pxp_settinguptaxonomy_windowappear.PNG
|
|

.. image:: /feature/_images/mdmimages/pxp_settinguptaxonomy_windowappear2.PNG
|
|

Icons can also be added to the Taxonomy.

    * Click a section under taxonomies (in this demo, we select **Casual Clothing**).

    .. image:: /feature/_images/mdmimages/pxp_settinguptaxonomy_clicksection.PNG
    |
    |

    * Click on **Upload** icon and select an image from your computer.

    .. image:: /feature/_images/mdmimages/pxp_settinguptaxonomy_clickuploadicon.PNG
    |
    |

    * View the selected icon.

    .. image:: /feature/_images/mdmimages/pxp_settinguptaxonomy_viewicon.PNG
    |
    |

|
