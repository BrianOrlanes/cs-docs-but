========================
Manage Virtual Catalogs
========================
The Manage Virtual Catalogs feature creates multiple collections and bookmarks to effectively manage different selections of products in the catalog.


===================     =======================================================================================================================================================================
**Key Term:**           * **Class:** a class defines the base configuration for products.

**Description:**        * A *Virtual Catalog* is a collection of various items put together by a user (usually a retailer) to be used at a specific time or for a certain occasion.
                        * The catalogs vary according to users and access to these catalogs can be restricted in terms of who can view and edit them.

**Related topics:**     * :doc:`Collections and Bookmarks </feature/product_feature/mdm/general/collectionsandbookmarks/index>`

**Application:**        * A retailer uses a *Virtual Catalog* to create his own collection of products for special occasions.
                        * Catalogs can be made user-specific.
                        * *Example:*

                            *Retailer A* with a *clothing franchise* creates a winter collection and uses this virtual catalog when the winter season arrives.

                        * *Note: The collection created is not in PIM but in the entity, Virtual Catalog*.

===================     =======================================================================================================================================================================


How-to: Manage Virtual Catalogs
================================
To do this feature, follow these steps:

|

* Log in and click on the **Administration** icon.

.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_login.PNG
|
|


* In **Data Model**, go to *Classes* and click on **Virtual Catalog**.

.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_clickvirtualcatalog.PNG
|
|


* Click on the '+' icon to create a new virtual catalog.

.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_clickplus.PNG
|
|


* Enter the name, enable as nature type, and select the type.
  Click on the **Create** button.

.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_entername.PNG
|
|


* Once the pane on the right of the screen has loaded, complete the remaining details.

.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_completedetails.PNG
|
|


* Add a *Property Collection* by clicking the '+' icon in *Section*. Select a *Property Collection* and then click the **Apply** button.

.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_addapropertycollection.PNG
|
.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_addapropertycollection2.PNG
|
|


* Click on the **Explore** icon and go to *Virtual Catalog*.

.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_clicktheexplorebutton.PNG
|
.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_clicktheexplorebutton2.PNG
|
|


* Click on the **Create** button and select the virtual catalog.

.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_clickcreatebutton.PNG
|
.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_clickcreatebutton2.PNG
|
|


* In the *Overview* tab, click on the **Add Product(s)** button under the *Products* section.

.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_clickaddproducts.PNG
|
|


* Create a collection by adding products (or selecting them from existing products).
  Choose the products to be added in the collection and click on the **Add** button.

.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_createacollection.PNG
|
.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_createacollection2.PNG
|
.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_createacollection3.PNG
|
|


* Click on the **Back** icon. View and check the virtual product catalog.

.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_viewandcheck.PNG
|
.. image:: /feature/_images/mdmimages/pxp_managevirtualcatalog_viewandcheck2.PNG
|
|
