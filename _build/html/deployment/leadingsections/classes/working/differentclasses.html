Different Types of Classes¶

Article¶

Article refers to the product in the system. It refers to the product which is a sellable item. This type of class is used to configure different types of products/ articles in the system. It defines the behavior and characteristics of the type of product. It defines the blueprint of how a product

Example:

We want to define the attributes and characteristics of a single article in the system. We can create a class which contains the attributes and characteristics of a single article, and then we can use this class to create multiple single articles in the system.

Asset¶

Asset refers to the digital assets of the system which contains media assets such as images, videos, documents, files, and so on. This type of class is used to configure different types of digital assets in the system like image, video, document, file, and so on. It defines the behaviour and characteristics of any type of digital asset. It defines the blueprint of how a digital asset should look like in the system.

Example

We want to define the attributes and characteristics of an image in the system. We can create a class which contains the attributes and characteristics for an image, and then we can use this class to create multiple images in the system.

Campaign¶

Campaign is an activity designed to promote and increase the awareness of your business and its products. This type of class is used to configure the marketing campaign. It defines the behaviour and characteristics of the product for campaigning purpose. When you want to promote your product for a longer duration, it is advisable to create the campaign in your system using this class.

Example

A furniture manufacturer promotes its newly introduced dining set. He wants to increase the awareness of the newly introduced dining set among his customers. He will promote this product by giving 10% discount on the dining set for a month. He can create the campaign class in the system and can promote the dining set.

Promotion¶

Promotions refer to the entire set of activities, which communicate the product, brand or service to the user. The idea is to make people aware, attract and induce to buy the product, in preference over others. This type of class is used to configure the promotion activity. It defines the behaviour and characteristics of the promotion. Promotion is for shorter duration and it helps to increase the product sales instantly.

Example

A furniture manufacturer promotes its newly introduced lounge chair. He wants to increase the awareness and sales of the newly introduced lounge chair. He will promote this product by giving 10% discount on the lounge chair on Mother’s Day. He can create the promotion class in the system and can promote the product.

Supplier¶

Supplier is any organization that uses the system. This type of class is used to manage the supplier details. It gives the insight of all the critical activities related to all the suppliers in the system. It defines the behaviour and characteristics of the supplier.

Example

We need to add the details of different types of vendors and suppliers in the system. We need the insight of all the critical activities related to all the suppliers in the system. A user can create the supplier class which describes the characteristics and behaviour of the suppliers and he can use this class to create multiple suppliers in the system.

Target¶

Target refers to a group of consumers to which company wants to sell its product. It refers to the market segment which consists of the customers whose needs and desires most closely match your product. This type of class is used to manage the target details. It defines the behaviour and characteristics of the targets.

There are three types of targets:

  • Market: This type of target refers to the international super market chains or retail stores.

  • Persona: Persona refers to an individual customer to which company wants to promote and sell its product considering his likes, dislikes, and interest area.

  • Situation: Situation target refers to an individual customer to which company wants to promote and sell its product after evaluating and analysing the current situation of his life. Situation can be based on facts, opinions, beliefs, haunches, and feelings.

    Example A furniture manufacturer wants to add multiple retail stores’ details in the system as his target market. He will create the Market class, in which he will define the characteristics and behaviour of the target market. He will then add multiple retail store details in the system using this class.

Text Asset¶

Text Asset manages the text or content in the system. It is helpful in defining he format of text or any type of content in the system. Text Asset class is used to manage any type of content in the system. It gives the insight of how the content should look like in the system, what should be the format, how to manage paragraphs or any type of text in the system. It creates the blue print of text or content in the system and then it can be used to present different types of content in the system.

Example

A furniture manufacturer wants to add ’Product Description’ and ‘About US’ details in the system. He will create a Text Asset class, in which he will define the paragraph style, font style, size, color, margins, and other details related to text. Now, he can use this class each time he needs to add any details or content regarding the product.
