Product Template¶

A product template may consist of one or more Adobe © InDesign © frames that can be directly associated with attribute placeholders within the frame or on the frame (e.g., text, image, table). Placement options (e.g., Fit content to frames, Fit frames to content), and paragraph styles (including GREP styles) are assigned to the frame via Adobe © InDesign © functionality. The frames of a product template can be placed on different levels. By saving and naming a product template, it is read into the PXP system. The value for the entire length and width of the elements of the basic template is determined (size of the rectangle enclosing all frames). When enriching the template in the PXP system (outside of Adobe © InDesign ©), the following information is collected:

Size (length, width) Initially filled Distribution Channels (VTS): defines as to which distribution channels the template can be used (see distinction/tagging) Assortment Validity: defines as to which assortment ranges the template can be used (see distinction/tagging)


Purpose:
  • This document will help you create ‘Product Template’ in Adobe © InDesign ©.
Target Audience:
  • Template Planners
  • Managers
  • Product Owners

How-to: Create Product Templates in Adobe © InDesign ©¶

  • Open the Adobe © InDesign © application.

  • Navigate to the Window, scroll down to Extensions and click on CS Desktop Publisher.

  • On the pop-up window, fill in the details like Configuration Presets, Server URL, User Name, and Password.

  • Append the server URL with extension dtp.

  • Click on the Save button to save the information or simply proceed to Login.

  • Click on PT to create a new product template.

  • Click on the Create template icon.


How-to: Add Placeholder Frames for Images¶

  • Click on the Rectangular Tool to place frames on the sheet. The frames serve as placeholders for the text and images in the template.

  • To place graphic images, click on the Rectangular Frame tool icon then click and drag a frame on the template.


How-to: Add Placeholder Frames for Attributes¶

  • Similarly, place text frames as a placeholder for attributes.


How-to: Link Attributes/Assets to a Placeholder in InDesign¶

To link the attributes of the CS system, select the frame (or text) then click on the Attributes. Search and select the respective attribute then click on the Link button at the bottom of the screen.


You can also link images to the template by doing the following steps:

  • Select the frame that you want to use as placeholder of the image.

  • Click on the Images icon in the CS Desktop Publisher window to see the list of available images in the system.

  • Select a suitable image and click on Link.

  • The image appears in the selected frame. To fit an image based on the frame size, right-click on the image and select the appropriate option.

  • Click on the Save icon to save the template. Please make sure you are in the product template section while saving.


How-to: Update/Save Product Templates in Adobe © InDesign ©¶

  • Click on the Save icon and on the pop-up window, enter a name for the template and click on the Save button.

How-to: Manage Product Templates in Contentserv PXP application¶

  • After saving the product template, switch to the Contentserv PXP application. Click on the user profile and select MxM.

  • Click on the Templates menu to see the product template you have just created.

  • Add content to the product template.

  • Click on the Save button.
