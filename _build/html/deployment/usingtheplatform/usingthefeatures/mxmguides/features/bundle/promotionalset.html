Promotional Sets¶

Promotional Set is a feature that includes a combined group of Marketing products, which can be used by System Administrators, Marketing Professionals, and Analysts who creates the promotional sets.

Purpose:
  • The purpose of this document is to explain what is a promotional set and how to configure it in a system. You will be guided through the different stages that are involved in the configuration of a promotional set.
Target Audience:
  • System Administrators
  • Marketing Professionals
  • Analysts who will create promotional sets
Prerequisites
  • Valid credentials to access Contentserv PXP system.
  • More than one Marketing Products must exist in the system.
  • Marketing Products must contain unit tags.
Overview:
  • The promotional set is a collection of multiple Marketing Products, which are put together to form a set. It offers several products for sale as one combined product. The set may contain the similar products with same sales unit and common sales price. The user can choose to buy any of the Marketing Product from the set at the defined common price. It is used for promotion and marketing of your business products. A promotional set is used for any special offer intended to raise a customer’s interest and influence a purchase. For example, a retailer offers the promotional set that contains three different types of cold drink bottles of 1 liter. Here, a retailer has set a common sales unit, which is the 1-liter bottle. He will set a common sales price for this promotional set. A customer can buy any of the cold-drink from a set at a defined common sales price.
Key Terms:
  • Marketing Product: The Marketing Product represents the marketing and promotional sales aspect of MDM product. It is virtually composed of MDM article along with the marketing and sales information. Overview of Promotional Set.


How-to: Configure the Promotional Set¶

To do this, follow these steps:

  1. Log into Contentserv and click on the Administration icon.


  1. Navigate to Data Model → Classes → Marketing Content → Marketing Bundle menu.


  1. Click on the plus icon to create a marketing bundle.


#. A pop-up window appears, enter the Name and Code. Enable the radio button for Create class as nature type field. Select the Promotional Set option from Type. Select created marketing product class from Nature Class field, for which you want to create a promotional set Click on Create button.



  1. Click on the Explore icon to display the Dashboard


Create a Promotional Set Containing Marketing Products¶

To do this, follow these steps:

  1. Click on the Admin icon and select MxM option from the drop-down list.


  1. Navigate to Bundles menu on left side panel.


  1. Click on Create drop-down and select the created Promotional Set class to make the set of Marketing Products.


#. It will redirect you to a window from where you can select the multiple products to add them to a promotional set. Select the products and click on icon to add the products into a setand click Add.



  1. You will see that selected products are added on the right side. Click on Select icon and click on Next.


  1. You will see the filters based on the units of different products. You will see the availability of selected products in different sales units.


#. Select a particular unit to make a promotional set of selected products pertaining to a unit. This means that you will make a promotional set of products in the same sales unit. Once you select the unit, click on Create button to create a promotional set.



  1. See the created promotional set in the list.


Promotional Set in Content Open View¶

To do this, follow these steps:

  1. Click on the created promotional set and see the content open view.


  1. Enter a name for a promotional set by clicking on the Edit icon.


  1. In the Overview tab of a content, unitthe created unit. This will be a promotional set of products for this unit.


  1. See all promotional set products in the Marketing Products section.


  1. In the Overview tab, select any product from the Marketing Products section as a father article.


  1. Click on the Save button once all the changes in the set are made.
