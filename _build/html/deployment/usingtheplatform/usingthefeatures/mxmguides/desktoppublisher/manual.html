CS Desktop Publisher Manual¶

CS Desktop Publisher and Adobe InDesign ©¶

The CS DTP is implemented based on the standard Adobe InDesign © functionality. The following InDesign features and functions are supported:

  • Formatting
  • Character and paragraph styles
  • Text frame options: Placing an image will adjust the frame to the context (image placement behavior in order to Adobe © InDesign © graphic frame settings).
  • GREP-Style
  • Adobe © InDesign © Library

How-to: Start CS Desktop Publisher¶

  • Open Adobe © InDesign © software from your system.

  • Navigate to Windows → Extension → CS Desktop Publisher.

  • A new pop-up window appears on the screen.

  • On the pop-up window, enter the Server URL. It is the URL of your Contentserv application.

  • Enter the username and password that you use to log in to the Contentserv application.

  • Click on the Save button to save the configuration details and then click on the Login button.
