=====================
Create a New Product
=====================
The system allows effective management of products in the system. Products can be created, deleted, and edited.


How-to: Create New Products
==============================
To do this, follow these steps:

|

*	Log in and go to *Products*.

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_clickexplore.PNG
|
|


* Click on the **Create** button and select the desired product (in this demo, we select **Product**).

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_clickcreate.PNG
|
|

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_clickcreate2.PNG
|
|

* Enter the necessary information.

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_enternecessaryinfo.PNG
|
|

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_enternecessaryinfo2.PNG
|
|

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_enternecessaryinfo3.PNG
|
|

* Go back to *Overview* tab and click on **Add Taxonomy** to add taxonomy to the product.

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_gobackoverview1.PNG
|
|

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_gobackoverview2.PNG
|
|

* Taxonomy menu appears. Select the desired taxonomy (in this demo, we select **Mobile Phones**).

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_taxmenuappears.PNG
|
|

* View the selected taxonomy.

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_viewselectedtax1.PNG
|
|

* Click on the **Relationships** tab to add an image to the product.


.. image:: /feature/_images/mdmimages/pxp_createnewproduct_clickrelationshiptab.PNG
|
|

* Click on the '+' icon to add an image.


.. image:: /feature/_images/mdmimages/pxp_createnewproduct_clickplus.PNG
|
|

* Click on the checkbox to add the desired image (button turns to blue when selected), then click on the **Add** icon.

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_clickselect.PNG
|
|

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_clickselect2.PNG
|
|

* Another way to add an image is to drag an image to the right pane.

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_dragimage.PNG
|
|

* To add more images, click on the **Back** icon, and repeat the previous steps.

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_addimageclickback.PNG
|
|

.. image:: /feature/_images/mdmimages/pxp_createnewproduct_addimageclickback2.PNG
|
|

|
